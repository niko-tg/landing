Чтобы почитать о команде, например о команде мут пиши команда мут

Список команд можно просмотреть введя - команды

Бан

Блокировка пользователя в чате

бан

↳ Банит пользователя, ответом на его сообщение (перманентно)


бан 123456789 12ч

↳ Банит по ID на 12 часов (без срока — перманентно)

Разбан

Снять бан с пользователя

разбан

↳ Снимает бан, ответом на сообщение


разбан 123456789

↳ Снимает бан по ID

Мут

Запрет писать в чат

мут 30м ↳ на 30 минут

мут 4ч ↳ на 4 часа

мут 1д ↳ на сутки

мут ↳ по умолчанию на 5 минут


По ID: мут 123456789 12ч

Анмут

Разрешить участнику писать в чат

анмут

↳ Снимает мут, ответом на сообщение


анмут 123456789

↳ Снимает мут по ID

Репорт

Пожаловаться модераторам

Ответь на сообщение нарушителя и напиши репорт причина (или жалоба).

Жалоба уйдёт в модераторский чат / админам.

Задать приветствие

Приветствие новых участников чата

Ответь командой /mkhello на сообщение, которое должно стать приветствием.

В тексте можно использовать ${user} (упоминание новичка) и ${chat} (название чата).


⚠️ Приветствие отправляется, только если оно включено в настройках чата (/settings).

Удалить приветствие

Удалить приветственное сообщение — /rmhello

Показать приветствие

Показать текущее приветствие — /gethello

Настройки чата

Интерактивные настройки чата (кнопками)

Открыть — /settings


Можно включить/выключить: модераторские команды, антифлуд, капчу, приветствие новичкам.

⚠️ Часть настроек (удаление ссылок и пересылок, бан от имени канала) доступна только в чатах с VIP-владельцем.

Журнал модерации

Журнал действий модерации (баны, муты и пр.) — /modlog

Синхронизация стаффа

Пересинхронизировать состав администраторов чата из Telegram — /reload

Полезно, если недавно поменяли админов и бот ещё не видит изменения.
